Инструкция к запуску:
1)В файле Main.py передать значения переменной "a" в виде [(3, 5), [((2, 2), 1)], [((3, 2), 1), ((2, 2), 2)]](взято из примера в задании)
2)запустить
p.s нужна библиотека NumPy команда для установки(pip install numpy)
-------------------------------------------------------------------
Комментарии, относящиеся к логике
l_model_left и l_model_right возвращают список для построения L элементов
build_L проверяет возможность построения L элемента. Возвращает список булевых значений
k_model возвращают список для построения К(квадрат) элементов
-------------------------------------------------------------------
Принцип работы алгоритма
После построения К элемента искать ближайшие к нему пустые клеточки и проверять возможность построения там Л элемента
